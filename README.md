# Todo Kanban for Obsidian

一款 Obsidian 插件，核心解决「当日待办管理」和「历史待办可视化」问题，支持任务创建、完成状态勾选、按日期维度组织，所有数据本地存储，界面适配 Obsidian 原生交互逻辑与主题风格。

## 功能特性

### 当日待办看板
- **任务输入**：支持单行/多行任务输入，按 Enter 键添加，Shift+Enter 换行
- **附加信息**：添加任务时可选择附带链接和截止日期
- **看板展示**：分为「未完成」「已完成」两个分区，卡片式展示
- **交互操作**：点击勾选框切换完成状态，勾选后自动从当前筛选视图中移除
- **双击编辑**：双击任务内容可编辑任务信息（内容、链接、截止日期）
- **智能排序**：任务按截止时间排序，无截止时间的任务优先显示在最上面
- **任务管理**：支持任务完成状态切换、任务删除、任务移动

### 历史待办可视化
- **数据聚合**：按日期分组展示历史任务，今天在最上面
- **日期筛选**：支持筛选「当日待办」「近7天」「近30天」「本周」「全部日期」
- **状态筛选**：每个日期分组下支持「未完成/已完成」筛选，默认显示未完成任务
- **批量操作**：支持清空指定日期的所有任务
- **移动任务**：历史任务可一键移动到今天

### 智能继承
- **插件加载时**：自动将历史未完成任务继承到今天
- **定时任务**：每天凌晨1点自动将历史未完成任务继承到今天
- **避免重复**：智能检测已继承的任务，避免重复添加

### 搜索功能
- **关键词搜索**：支持搜索任务内容、链接、截止日期
- **实时过滤**：输入即搜索，实时更新任务列表
- **不区分大小写**：搜索时忽略大小写

### 导出功能
- **日期范围导出**：支持选择开始日期和结束日期导出
- **CSV格式**：导出为CSV格式，Excel可直接打开
- **完整信息**：导出内容包括日期、任务内容、状态、截止日期、链接、创建时间
- **中文支持**：完美支持中文内容导出

## 安装步骤

1. **下载插件**：将本项目文件夹复制到 Obsidian 库的 `.obsidian/plugins/` 目录下
2. **启用插件**：在 Obsidian 设置 → 社区插件中启用「Todo Kanban for Obsidian」
3. **重启 Obsidian**：重启后即可使用

## 使用指南

### 打开面板
- 点击 Obsidian 左侧边栏的「待办看板」图标（勾选框图标）打开待办面板

### 添加任务
1. 在底部输入框中输入任务内容
2. 可选：填写链接（支持任意URL）
3. 可选：选择截止日期
4. 按 Enter 键或点击「添加」按钮添加任务

### 编辑任务
- 双击任务内容区域，弹出编辑对话框
- 可修改任务内容、链接、截止日期
- 点击「保存」保存修改

### 筛选与搜索
1. **日期筛选**：在顶部下拉框中选择日期范围
2. **关键词搜索**：在搜索框中输入关键词实时搜索
3. **状态筛选**：在每个日期分组下选择「未完成」或「已完成」

### 导出任务
1. 点击顶部「导出Excel」按钮
2. 选择开始日期和结束日期
3. 点击「导出」下载CSV文件

### 其他操作
- **标记完成**：点击任务左侧的勾选框
- **删除任务**：点击任务右侧的「×」按钮
- **移动到今天**：历史任务可点击「→今天」按钮移动到今天
- **清空日期任务**：点击日期标题右侧的「清空」按钮

## 数据存储

任务数据存储在 Obsidian 库根目录的 `.obsidian/plugins/todo_kanban/tasks.json` 文件中，采用 JSON 格式存储，确保数据安全可靠。

### 数据结构
```json
{
  "tasks": [
    {
      "date": "2026-03-08",
      "createTime": "2026-03-08 10:00",
      "tasksList": [
        {
          "taskId": "unique-id",
          "content": "任务内容",
          "completed": false,
          "createAt": "2026-03-08 10:00",
          "link": "https://example.com",
          "dueDate": "2026-03-10"
        }
      ]
    }
  ]
}
```

## 技术说明

- 基于 Obsidian 官方 API 开发
- 使用 JavaScript 编写，无编译依赖
- 适配 Obsidian 浅色/深色主题
- 无第三方依赖，纯原生实现
- 支持 Obsidian v1.0.0 及以上版本

## 性能要求

- 插件启动时间 ≤ 100ms
- 历史任务数据量 ≤ 1000 条时无卡顿
- 操作响应时间 ≤ 50ms

## 注意事项

- 所有数据仅存储在本地，无网络请求
- 插件完全免费开源，无付费功能
- 请定期备份 `.obsidian/plugins/todo_kanban/tasks.json` 文件，以防数据丢失
- 历史未完成任务会在每天凌晨1点自动继承到今天
